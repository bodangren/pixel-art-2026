# Technology Stack - Pixel Art Benchmark

## Core Framework
- **Language:** [TypeScript](https://www.typescriptlang.org/) (Version ^5)
- **Frontend Framework:** [Next.js](https://nextjs.org/) (Version 16.2.2, App Router)
- **Rendering Library:** [React](https://react.dev/) (Version 19.2.4)

## Styling and Layout
- **CSS Framework:** [TailwindCSS](https://tailwindcss.com/) (Version ^4)
- **Post-processing:** [PostCSS](https://postcss.org/)

## Utilities and Data Validation
- **Validation Library:** [Zod](https://zod.dev/) (Version ^4.3.6)
- **Image Processing:** [Sharp](https://sharp.pixelplumbing.com/) (Version ^0.34.5)

## Development and Deployment
- **Runtime Environment:** [Node.js](https://nodejs.org/) (Version ^20)
- **Linting:** [ESLint](https://eslint.org/) (Version ^9)
- **Build Target:** Static Export (`output: 'export'`)
