# Pixel Art Asset Generation Prompt

You are an expert pixel artist specializing in game-ready sprite sheets and textures.

## Task
Generate a complete asset pack for the game **Labyrinth of the Goblin King**.

## Theme
A dark dungeon maze crawling with goblins. The player is a hero/knight who becomes a Paladin with a golden aura.

## Assets to Generate

### 1. Background
- **ID**: background
- **Description**: A dark, atmospheric stone dungeon floor texture suitable for a maze background.
- **Dimensions**: 1024x1024 pixels
- **Format**: PNG with transparency

### 2. Hero Sprite Sheet (3x3)
- **ID**: hero-3x3-sheet
- **Description**: A heroic knight character. 3x3 sprite sheet showing idle, walk, and attack poses.
- **Tile Size**: 64x64 pixels
- **Sheet Dimensions**: 192x192 pixels (3x3 grid)
- **Format**: PNG with transparency

### 3. Goblin Sprite Sheet (3x3)
- **ID**: goblin-3x3-sheet
- **Description**: A small, green, menacing goblin. 3x3 sprite sheet with idle, walk, and attack poses.
- **Tile Size**: 64x64 pixels
- **Sheet Dimensions**: 192x192 pixels (3x3 grid)
- **Format**: PNG with transparency

### 4. Orb Effect Sheet (1x3)
- **ID**: orb-sheet
- **Description**: Magical word orbs. 1x3 sheet showing standard and glowing states.
- **Tile Size**: 64x64 pixels
- **Sheet Dimensions**: 192x64 pixels (1x3 grid)
- **Format**: PNG with transparency

## Technical Requirements

1. **Pixel Art Style**: Use a limited color palette (max 16 colors per sprite)
2. **Grid Alignment**: All sprites must be perfectly aligned to an 8x8 or 16x16 pixel grid
3. **3x3 Sprite Sheets**: Use 3 columns (idle, walk, attack) x 3 rows (different animation frames)
4. **Transparency**: Use alpha channel for transparent areas (not a solid color key)
5. **Output**: Save as PNG files with descriptive filenames

## Color Palette Guidance
- Background: Stone grays (#2a2a2a, #3a3a3a, #4a4a4a) with darker cracks
- Hero: Steel grays with gold accents (#c0c0c0, #ffd700, #4169e1)
- Goblin: Greens and browns (#228b22, #556b2f, #8b4513)
- Orbs: Magical blues and purples (#4169e1, #9370db, #00ced1)

## Output Format
Save all PNG files to an `assets/` directory within your response workspace.
